EMChatEventHandler

EMChatEventHandler class

Chat 事件监听。

用于监听收消息，已读回执，等回调。

添加监听：

  EMClient.getInstance.chatManager.addEventHandler(UNIQUE_HANDLER_ID, EMChatEventHandler());

移除监听：

  EMClient.getInstance.chatManager.removeEventHandler(UNIQUE_HANDLER_ID);
