EMChatManager

EMChatManager class

聊天管理类，该类负责管理会话（加载，删除等）、发送消息、下载消息附件等。

比如，发送一条文本消息：

   EMMessage msg = EMMessage.createTxtSendMessage(
       targetId: toChatUsername, content: content);
   await EMClient.getInstance.chatManager.sendMessage(msg);
