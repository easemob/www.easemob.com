EMChatRoomManager

EMChatRoomManager class

聊天室管理类，负责聊天室加入和退出、聊天室列表获取以及成员权限管理等。 比如，加入聊天室：

  try {
      await EMClient.getInstance.chatRoomManager.joinChatRoom(chatRoomId);
  } on EMError catch (e) {
      debugPrint(e.toString());
  }
