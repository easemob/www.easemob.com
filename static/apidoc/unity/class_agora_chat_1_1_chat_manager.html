◆ DeleteConversation()

bool AgoraChat.ChatManager.DeleteConversation ( string  conversationId,
bool  deleteMessages = true 
)

删除本地数据库中的指定会话及其历史消息。

若将 deleteMessages 设置为 true，删除会话的同时也会删除该会话的本地历史消息。

参数
conversationId会话 ID。
deleteMessages是否同时删除本地历史消息：
  • true ：是；
  • false ：否。
isThread删除会话是否是子区会话。
  • true 是子区会话；
  • false 不是子区会话。
返回
会话是否成功删除：
  • true ：是；
  • false ：否。
