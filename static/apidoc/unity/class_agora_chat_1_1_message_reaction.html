详细描述

消息 Reaction 实体类，有如下属性：

Reaction：消息 Reaction。 UserCount：添加了指定 Reaction 的用户数量。 UserList：添加了指定 Reaction 的用户列表。 State 当前用户是否添加了该 Reaction。

◆ UserList

List<string> AgoraChat.MessageReaction.UserList

获取添加了指定 Reaction 的用户列表。

Note 只有通过 IChatManager#GetReactionDetail 接口获取的是全部用户的分页数据；其他相关接口如IChatManager#GetReactionList等都只包含前三个用户。
