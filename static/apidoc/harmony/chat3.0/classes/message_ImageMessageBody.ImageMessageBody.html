setThumbnailSize

  • setThumbnailSize(width, height): void

  • 设置缩略图的尺寸。

    图片消息发送到服务器后，服务器会根据此处设置的缩略图尺寸对原图进行压缩。

    如果没有设置，服务器会根据默认值进行压缩。

    Parameters

    • width: number

      指导服务器生成缩略图的宽度。

    • height: number

      指导服务器生成缩略图的高度。

    Returns void

    Remarks

    需要图片消息发送前设置。
