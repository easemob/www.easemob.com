canJoinDirectly

  • canJoinDirectly(): boolean

  • 获取用户是否能自由加入群组，还是需要申请或者被邀请。

    canJoinDirectly 是 GroupStyle.GroupStylePublicOpenJoin 类型群组的属性，表示该群可以自由加入的群组。

    Returns boolean

    群组属性：

    • true：用户可以自由加入群，不需要申请和被邀请。
    • false：进群需要群主、管理员或群成员邀请或者群主和管理员同意入群申请。
