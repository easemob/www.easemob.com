com.hyphenate.chat.EMCheckType Class Reference

Detailed Description

The Chat service diagnoses process, including account validation, whether the DNS LIST and token is got from the server, whether the user has logged in or logged out.

This enum lists the server diagnostic tests which are run for the current user when you call EMClient#check(String, String, EMClient.CheckResultListener). This is an asynchronous method. If there is an error, check the error code to know the error information.

Make sure not to update the existing data, because it is related to the native CheckType defined in emchatconfigs.h.
