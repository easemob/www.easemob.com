◆ createChatThread()

void com.hyphenate.chat.EMChatThreadManager.createChatThread ( String parentId,
String messageId,
String chatThreadName,
EMValueCallBack< EMChatThread > callBack )

创建子区。

子区所属群组的所有成员均可调用该方法。

子区创建成功后，会出现如下情况：

  • 单设备登录时，子区所属群组的所有成员均会收到回调 EMChatThreadChangeListener#onChatThreadCreated(EMChatThreadEvent)。 你可通过设置 EMChatThreadChangeListener 监听相关事件。
  • 多端多设备登录时，各设备会收到事件回调 com.hyphenate.EMMultiDeviceListener#onChatThreadEvent(int, String, List)。 该回调方法中第一个参数表示子区事件，例如，子区创建事件为 com.hyphenate.EMMultiDeviceListener#THREAD_CREATE。 你可通过设置 com.hyphenate.EMMultiDeviceListener 监听相关事件。
参数
parentId父 ID，即群组 ID。
messageId父消息 ID。
chatThreadName要创建的子区的名称。长度不超过 64 个字符。
callBack结果回调：
  • 成功时回调 EMValueCallBack#onSuccess(Object)，返回创建成功的子区对象；
  • 失败时回调 EMValueCallBack#onError(int, String)，返回错误信息。

参数
chatThreadId子区 ID。
callBack结果回调：
  • 成功时回调 EMCallBack#onSuccess()；
    • 失败时回调 EMCallBack#onError(int, String)，返回错误信息。

◆ joinChatThread()

void com.hyphenate.chat.EMChatThreadManager.joinChatThread ( String chatThreadId,
EMValueCallBack< EMChatThread > callBack )

加入子区。

子区所属群组的所有成员均可调用该方法。

多端多设备登录时，注意以下几点：

  • 设备会收到 com.hyphenate.EMMultiDeviceListener#onChatThreadEvent(int, String, List) 回调。
  • 该回调方法中，第一个参数表示子区事件，例如，加入子区事件为 com.hyphenate.EMMultiDeviceListener#THREAD_JOIN。
  • 可通过设置 com.hyphenate.EMMultiDeviceListener 监听相关事件。
参数
chatThreadId子区 ID。
callBack结果回调：
  • 成功时回调 EMValueCallBack#onSuccess(Object)，返回子区详情 EMChatThread，详情中不含成员数量；
  • 失败时回调 EMValueCallBack#onError(int, String)，返回错误信息。 重复加入会报错，错误码为 com.hyphenate.EMError#USER_ALREADY_EXIST。

退出子区。

子区中的所有成员均可调用该方法。

  • 你可通过设置 com.hyphenate.EMMultiDeviceListener 监听相关事件；
参数
chatThreadId要退出的子区的 ID。
callBack结果回调：
  • 成功时回调 EMCallBack#onSuccess()；
  • 失败时回调 EMCallBack#onError(int, String)，返回错误信息。
