Detailed Description

The message thread manager that defines how to manage message threads, including message thread creation, destruction, and member management.

)

Creates a message thread.

Each member of the chat group where the message thread belongs can call this method.

Upon the creation of a message thread, the following will occur:

  • In a single-device login scenario, each member of the group to which the message thread belongs will receive the EMChatThreadChangeListener#onChatThreadCreated(EMChatThreadEvent) callback. You can listen for message thread events by setting EMChatThreadChangeListener.
  • In a multi-device login scenario, the devices will receive the com.hyphenate.EMMultiDeviceListener#onChatThreadEvent(int, String, List) callback. In this callback method, the first parameter indicates a message thread event, for example, com.hyphenate.EMMultiDeviceListener#THREAD_CREATE for a message thread creation event. You can listen for message thread events by setting com.hyphenate.EMMultiDeviceListener.
Parameters
parentIdThe parent ID, which is the group ID.
messageIdThe ID of the parent message.
chatThreadNameThe name of the new message thread. It can contain a maximum of 64 characters.
callBackThe result callback:
  • If success, EMValueCallBack#onSuccess(Object) is triggered to return the new message thread object;
  • If a failure occurs, EMValueCallBack#onError(int, String) is triggered to return an error.

In this callback method, the first parameter indicates a message thread event, for example, com.hyphenate.EMMultiDeviceListener#THREAD_DESTROY for a message thread destruction event. You can listen for message thread events by setting com.hyphenate.EMMultiDeviceListener.

EMValueCallBack< EMChatThread >  callBack 
)

Joins a message thread.

Each member of the group where the message thread belongs can call this method.

In a multi-device login scenario, note the following:

  • The devices will receive the com.hyphenate.EMMultiDeviceListener#onChatThreadEvent(int, String, List) callback.
  • In this callback method, the first parameter indicates a message thread event, for example, com.hyphenate.EMMultiDeviceListener#THREAD_JOIN for a message thread join event.
  • You can listen for message thread events by setting com.hyphenate.EMMultiDeviceListener.
Parameters
chatThreadIdThe message thread ID.
callBackThe result callback:
  • If success, EMValueCallBack#onSuccess(Object) is triggered to return the message thread details EMChatThread which do not include the member count.
  • If a failure occurs, EMValueCallBack#onError(int, String) is triggered to return an error. If you join a message thread repeatedly, the SDK will return an error with the error code com.hyphenate.EMError#USER_ALREADY_EXIST.
