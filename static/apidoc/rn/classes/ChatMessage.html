    Class ChatMessage

    消息类，用于定义一条要发送或接收的消息。

    例如，构造一条文本消息：

      let msg = ChatMessage.createTextMessage(
            'asteriskhx2',
            Date.now().toString(),
            ChatMessageChatType.PeerChat
          );

    Properties

    attributes

    attributes: Record<string, any>

    消息的扩展属性。

    body

    body: ChatMessageBody

    消息体实例，详见 ChatMessageBody。

    chatType

    chatType: ChatMessageChatType = ChatMessageChatType.ChatRoom

    会话类型，包括单聊，群聊和聊天室。详见 ChatType。

    conversationId

    conversationId: string = ''

    会话 ID。

    deliverOnlineOnly

    deliverOnlineOnly: boolean

    消息是否只投递给在线用户：

    • true：只有消息接收方在线时才能投递成功。若接收方离线，则消息会被丢弃。
    • （默认）false：如果用户在线，则直接投递；如果用户离线，消息会在用户上线时投递。

    direction

    direction: ChatMessageDirection

    消息方向，详见 ChatMessageDirection。

    from

    from: string = ''

    消息发送者的用户 ID。

    groupAckCount

    groupAckCount: number = 0

    群聊时，已阅读消息的群成员数量。群成员已读消息后调用 ChatManager.sendGroupMessageReadAck 或者 ChatManager.sendConversationReadAck 发送已读回执。若需要开启已读回执功能，初始化时需设置 ChatOptions.requireAck 为 true， 并且发送消息时设置 isNeedGroupAck 为 true。

    hasDeliverAck

    hasDeliverAck: boolean = false

    单聊时，发送的消息是否送达至接收方。要使该参数生效，初始化时需接收方设置 ChatOptions.requireDeliveryAck 为 true。群消息不支持送达回执。

    • true：是。
    • （默认）false：否。

    hasRead

    hasRead: boolean = false

    单聊或群聊时，接收方是否已读了消息。该参数的值影响会话的未读消息数。阅读消息后，接收方可以调用 ChatManager.markMessageAsRead 或者 ChatManager.markAllMessagesAsRead 将消息设置为已读。

    • true：是。
    • （默认）false：否。

    hasReadAck

    hasReadAck: boolean = false

    单聊时，发送方是否收到了接收方的已读回执。接收方阅读消息后会调用 ChatManager.sendMessageReadAck 或者 ChatManager.sendConversationReadAck 发送已读回执。若需要已读回执，SDK 初始化时需设置 ChatOptions.requireAck 为 true。

    • true：是。
    • （默认）false：否。

    isBroadcast

    isBroadcast: boolean

    是否是广播消息。

    isChatThread

    isChatThread: boolean

    消息是否为子区消息：

    • true：是。你需将消息接收方的用户 ID 设置为子区 ID。详见 to。
    • false：否。

    注意

    该参数仅对群聊有效。

    isContentReplaced

    isContentReplaced: boolean

    消息内容是否被替换过。

    该属性仅在 ChatOptions.useReplacedMessageContents 开启时有效。

    isOnline

    isOnline: boolean

    消息是否为在线时收到的消息。

    • true: 是。这种情况下，如果应用在后台运行可以弹出消息提示框。
    • false: 否，离线时收到的消息。

    localMsgId

    localMsgId: string = ''

    本地消息 ID。

    localTime

    localTime: number = ...

    消息创建的本地 Unix 时间戳。单位为毫秒。

    msgId

    msgId: string = ...

    服务器生成的消息 ID。

    needGroupAck

    needGroupAck: boolean = false

    群聊时，是否需要消息已读回执。

    • true：是。
    • （默认）false：否。

    OptionalreceiverList

    receiverList?: string[]

    消息接收对象列表。

    默认为 undefined，表示群组或聊天室中的所有成员均能收到该消息。

    该属性只在群组或者聊天室中使用。

    serverTime

    serverTime: number = ...

    服务器收到该消息的 Unix 时间戳。单位为毫秒。

    status

    status: ChatMessageStatus

    消息发送状态，详见 ChatMessageStatus。

    OptionalstreamChunk

    streamChunk?: ChatStreamChunk

    消息流分片信息。

    to

    to: string = ''

    消息接收方的用户 ID：

    • 单聊：消息接收方的用户 ID；
    • 群聊：群组 ID；
    • 聊天室：聊天室 ID；
    • 子区：子区 ID。
