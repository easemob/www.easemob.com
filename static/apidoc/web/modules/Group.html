createGroupVNext

  • createGroupVNext(this: Connection, params: { allowMemberToInvite?: boolean; avatar?: string; description?: string; extension?: string; groupName?: string; inviteNeedConfirm?: boolean; isPublic?: boolean; maxMemberCount?: number; members?: string[]; needApprovalToJoin?: boolean }): Promise<AsyncResult<{ groupId: string }>>

  • 创建群组。

    connection.createGroupVNext({
     groupName: 'groupname',
        avatar: 'group avatar',
     description: 'this is my group',
     members: ['user1', 'user2'],
     isPublic: true,
        needApprovalToJoin: false,
        allowMemberToInvite: true,
        inviteNeedConfirm: false,
        maxMemberCount: 200,
        extension: JSON.stringify({info: 'group info'})
    })

    Parameters

    • this: Connection
    • params: { allowMemberToInvite?: boolean; avatar?: string; description?: string; extension?: string; groupName?: string; inviteNeedConfirm?: boolean; isPublic?: boolean; maxMemberCount?: number; members?: string[]; needApprovalToJoin?: boolean }
      • Optional allowMemberToInvite?: boolean

        是否允许群成员邀请其他人进群。true：允许；false：不允许。

      • Optional avatar?: string

        群组头像。

      • Optional description?: string

        群组描述。

      • Optional extension?: string

        群组详情扩展，可以采用 JSON 格式，以包含更多群信息。

      • Optional groupName?: string

        群组名称。

      • Optional inviteNeedConfirm?: boolean

        受邀人加入群组前是否需接受入群邀请。

        • true ：需受邀人确认入群邀请。默认值为 true。
        • false：受邀人直接加入群组，无需确认入群邀请。
      • Optional isPublic?: boolean

        是否为公开群，公开群：其他人可以调用 getPublicGroups 查询到的群组。- true：公开群；- false：非公开群。

      • Optional maxMemberCount?: number

        群组最大成员数。

      • Optional members?: string[]

        群成员 ID 数组，这些用户会被直接加入群组，在 onGroupEvent 回调里收到 operation:'directJoined'。

      • Optional needApprovalToJoin?: boolean

        加群是否需要管理员审批。- true：需要；- false：不需要。

    Returns Promise<AsyncResult<{ groupId: string }>>

  • getJoinedGroups(this: Connection, params: { needAffiliations?: boolean; needRole?: boolean; pageNum: number; pageSize: number; error?: any; success?: any }): Promise<AsyncResult<BaseGroupInfo[] | GroupInfo[]>>

  • 获取用户加入的所有群组。

    note

    若设置了 needAffiliations 或 needRole 参数为 true，分页获取时，当前页码 （pageNum）从 0 开始，每页获取的群组上限（pageSize）为 20 并且 返回值为类型为 Promise<AsyncResult<GroupTypes.GroupInfo[]>

    若不设置这两个参数，分页获取时，当前页码 （pageNum）从 1 开始，每页获取的群组上限（pageSize）为 500 并且 返回值为类型为 Promise<AsyncResult<GroupTypes.BaseGroupInfo[]>

    connection.getJoinedGroups({
            pageNum: 1,
            pageSize: 500,
            needAffiliations: false,
            needRole: false
    })
