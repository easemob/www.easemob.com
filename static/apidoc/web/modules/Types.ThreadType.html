RemoveMemberResult

RemoveMemberResult: { result: boolean; user: UserId }

子区成员移除结果定义。

Type declaration

  • result: boolean

    子区成员是否成功移除。

    • true：成功；
    • false：失败。
  • user: UserId

    子区成员 ID。

onChatThreadChangeType

onChatThreadChangeType: "create" | "update" | "destroy" | "userRemove"

子区事件类型定义:
create：当创建一个子区时触发。
update： 当子区内有新消息时触发。
destroy： 当删除一个子区时触发。
userRemove： 当你被从子区踢出时触发。
